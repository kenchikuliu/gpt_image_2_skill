## Loading policy

- Start here to choose a category; do not read the whole Scale into context.
- Read `craft.md` for general prompt-writing principles.
- Read exactly one category file for normal requests; read two or three only when the user asks for hybrid styles.
- Preserve `Original` versus `Author + Source` metadata when adapting examples into README/gallery entries.
- If entries move, update both this index and the corresponding category file in the same PR.
